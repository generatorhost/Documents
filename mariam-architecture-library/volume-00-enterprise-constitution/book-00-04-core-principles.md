Version: v1.0-draft  
Status: Started

## 1. Purpose

This book defines the permanent core principles governing every part of Mariam AI Enterprise OS.

## 2. Core Principles

1. One architecture library governs the project.
2. Every system is traceable.
3. Every runtime has a defined purpose.
4. Every agent has a role, mission, policy, capability, and DNA.
5. Every output passes quality and governance layers.
6. Every mission creates knowledge.
7. Every improvement is captured as controlled evolution.
8. Every domain uses reusable capabilities.
9. Every integration is managed through adapters, providers, connectors, or services.
10. Every release is measurable and reviewable.

## 3. Operating Rule

```text
Architecture → Blueprint → Specification → Implementation → Testing → Acceptance
```

## 4. Acceptance Criteria

This book is accepted when the system-wide principles can guide all future documentation and implementation.
