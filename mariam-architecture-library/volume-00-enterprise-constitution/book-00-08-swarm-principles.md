## 1. Purpose

This book defines swarm principles for large-scale parallel work inside Mariam AI Enterprise OS.

## 2. Swarm Role

Swarms are mission-focused groups of agents created to execute work in parallel under coordination, governance, review, and measurable objectives.

## 3. Swarm Principles

1. Every swarm has a mission.
2. Every swarm has a coordinator.
3. Every swarm has measurable outputs.
4. Every swarm uses approved agents and tools.
5. Every swarm reports progress, blockers, and results.
6. Every swarm can be scaled horizontally.
7. Every swarm can be decomposed into teams and sub-swarms.
8. Every swarm includes recovery and reassignment rules.
9. Every swarm output passes review and validation.
10. Every swarm execution can create learning artifacts and evolved DNA proposals.

## 4. Swarm Flow

```text
Chief
→ Mission
→ Department
→ Team
→ Swarm
→ Agent Tasks
→ Review
→ Delivery
```

## 5. Acceptance Criteria

This book is accepted when swarm operation, coordination, scaling, and recovery principles are clear.
