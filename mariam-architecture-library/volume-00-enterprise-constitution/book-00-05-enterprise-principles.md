Version: v1.0-draft  
Status: Started

## 1. Purpose

This book defines how Mariam behaves as an autonomous enterprise structure.

## 2. Enterprise Structure

Mariam is organized as:

```text
Enterprise
→ Board / Governance
→ Chief
→ Departments
→ Teams
→ Swarms
→ Agents
→ Tools / Models / Services
```

## 3. Enterprise Principles

1. Every department has a defined responsibility.
2. Every team serves an approved mission area.
3. Every swarm is created for measurable work.
4. Every agent acts under policy and permission.
5. Every output has accountable ownership.
6. Every business domain maps to workflows and capabilities.
7. Every domain can grow through DNA without changing the enterprise identity.
