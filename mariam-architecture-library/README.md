# Mariam Architecture Library

This repository folder is the official documentation workspace for **Mariam AI Enterprise OS**.

## Purpose

Mariam Architecture Library is the source of truth for the project architecture, planning, blueprint, terminology, engineering standards, specifications, development guide, operations guide, and testing guide.

All implementation work must trace back to this library.
